Git Branch and Merge

Using Command Line

Branching (Update Local Repo)

  1. cd into folder with local repository
  1. $ git branch will follow the remote branch with latest code
    • $ git checkout [branch name] if local
    • $ git checkout --track origin/[branch name] if not local
  1. $ git pull is a combo of fetch/pull from remote to update local
  1. $ git log shows logs/notes/messages for historical changes

Start Own Branch

  1. q will quit the local repo update and return to command prompt
  1. $ git branch [new branch name] (with no spaces, - only)
  1. $ git branch
  1. $ git checkout [new branch name]
  1. $ git push origin [new branch name] pushes new branch to remote

Delete Branch

  1. $ git branch -D [branch name] deletes local branch. Need to ensure you've use $ git checkout [name] to checkout another branch or main before you delete.
  1. Delete remote branch by logging into your remote host (ex. BitBucket, GitHub) and manually delete the branch.
