General Command Line

Click to: Learn Command Line

Command Description Notes
$ cd directoryname Change directory You can move multiple directories by entering the path. Ex. directory1/directory2
$ cd.. Move back up a directory You can move up multiple directories by repeating the command. Ex. $ cd../..
$ ls List files in your current directory
  • Type $ ls -a to view hidden directories
$ touch filename Will create a new file in your current directory. ex. $ touch test.py
$ mv Move a file from one directory to another, or move a directory.
  • File ex. $ mv NAMEOFFILE WHERETOMOVEIT
  • Directory ex. % mv NAMEOFFILE NAMEOFDIRECTORY from the directory to move it to.
  • You can move back with the ../ command
$ cp Copy a file. ex. $ cp NAMEOFFILETOCOPY NAMEOFNEWFILE
$ rm Delete files or directories ex. $ rm FILENAME
$ mkdir Make a new directory in your current directory location. ex. $ mkdir test

Git

Command Description Notes
$ git init Starts the local repository. Start from your top level project folder
$ git status Shows the status of your current project.
$ git add [file] or . Adds files to be tracked (Note: add a space before the .) Using . will grab all files in your project. Add a space before the .
$ commit [file] Commits the changes files - use when you've hit a good state. Generally, you'll want to commit all files and add a message, so you can use. $ git commit -a -m "Message"
$ git push Will push everything from your master to your cloud git repository. You must first setup your repository in BitBucket, Github, Gitlab, etc.
$ rm -f ./.git/index.lock Unlock Git. Use if you receive a message that Git is unable to push due to a lock.
