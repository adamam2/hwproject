Setup and Environment

Using Command Line

Initialize Virtual Environment

  1. cd into the top level project folder with your project's manage.py file
  2. Activate the virtual environment $ source venv/bin/activate
  3. Start Local Server $ python manage.py runserver

Initialize Git

  1. $ git init starts the local repository (Note: start from your top level project folder)
  2. $ git status shows the status of your current project
  3. $ git add [file] or . adds files to be tracked (Note: add a space before the .)
  4. $ git commit [file] commits the changes files - use when you've hit a good state. Generally, you'll want to commit all files and add a message, so you can use $ git commit -a -m "Message" to do that
  5. $ git push will push everything from your master to your cloud git repository (Note: You must setup first in Github, Gitlab, etc.)

Unlock Git

  1. $ rm -f ./.git/index.lock
